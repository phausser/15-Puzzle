# 15-Puzzle

A simple 15-puzzle game (sliding puzzle) built with HTML, CSS, and JavaScript.

## Description

The 15-puzzle is a classic sliding puzzle that consists of a frame of numbered square tiles in random order with one tile missing. The object of the puzzle is to place the tiles in order by making sliding moves that use the empty space.

## How to Play

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Click on the tiles to move them into the empty slot until they are sorted numerically.
